# Как войти в Instagram и создать сессию для агента

## Когда это нужно делать

- Первый запуск агента (нет ig_session.json)
- Агент пишет в Telegram: «нет сессии Instagram»
- Агент пишет: «сессия устарела / LoginRequired»
- После длительного простоя (сессия живёт ~30–90 дней)

---

## Шаг 1 — Подключи Mac к iPhone Hotspot

Instagram блокирует серверные IP. Нужен домашний/мобильный интернет.

1. На iPhone: **Настройки → Режим модема → Включить**
2. На Mac: подключись к Wi-Fi сети iPhone (называется как твой iPhone)
3. Проверь IP на [whatismyip.com](https://whatismyip.com) — должен быть мобильный (не серверный)

---

## Шаг 2 — Запусти скрипт на Mac

Открой Терминал и выполни:

```bash
cd ~/Documents/Claude/Projects/Автономный\ агент
python3 make_session.py
```

Скрипт сам установит зависимости, если нужно.

---

## Шаг 3 — Следуй инструкциям скрипта

**Сценарий A — всё прошло гладко:**
Увидишь `✅ СЕССИЯ СОЗДАНА УСПЕШНО` и путь к файлу.

**Сценарий B — Instagram требует SMS/Email код:**
1. Скрипт спросит: «1 — SMS, 2 — Email»
2. Выбери и введи код

**Сценарий C — Instagram требует 2FA код:**
1. Открой приложение-аутентификатор (Google Authenticator и т.п.)
2. Введи 6-значный код

**Сценарий D — IP всё равно заблокирован:**
1. Отключись от Hotspot
2. Подожди 5 минут
3. Снова подключись к Hotspot iPhone
4. Повтори шаг 2

---

## Шаг 4 — Скопируй сессию на Windows-сервер

После успешного логина файл `ig_session.json` появится в папке проекта на Mac.

### Вариант A — через GitHub (рекомендуется)

В Терминале на Mac:

```bash
cd ~/Documents/Claude/Projects/Автономный\ агент
python3 push_session.py
```

Затем на Windows-сервере:

```cmd
cd C:\InstAgent
python update.py
```

### Вариант B — скопировать вручную

Найди файл в Finder:
`~/Documents/Claude/Projects/Автономный агент/ig_session.json`

Загрузи его на сервер через FileZilla, WinSCP или любой FTP-клиент в папку `C:\InstAgent\`.

---

## Шаг 5 — Перезапусти агентов на сервере

На Windows-сервере:

```cmd
taskkill /f /im python.exe /t
cd C:\InstAgent
start.bat
```

Через ~1 минуту Telegram-бот пришлёт: `✅ DM-агент: сессия восстановлена`

---

## Признаки того, что сессия скоро устареет

- Агент пишет ошибки `LoginRequired` в Telegram
- Публикации перестали выходить в 09:00
- В аналитике в 20:00 нет данных по подписчикам

**Обычно сессия живёт 30–90 дней.** Можно продлить, не перелогиниваясь — агент сам обновляет сессию при каждом входе в Instagram.

---

## Частые проблемы

| Ошибка | Причина | Решение |
|--------|---------|---------|
| IP blacklisted | Mac подключён к офисному Wi-Fi или VPN | Переключись на iPhone Hotspot |
| Bad password | Неверный пароль в .env | Проверь IG_PASSWORD в файле .env |
| Challenge required | Instagram подозревает бота | Войди вручную в браузере, подожди 1 час |
| Сессия не работает на сервере | Сессия привязана к IP Mac | Нормально — агент логинится заново при старте |
